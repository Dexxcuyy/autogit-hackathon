Build a clean, responsive loan installment calculator app (KPR/KKB) using React + Vite + Tailwind CSS.

The app must have:
- Input fields: loan amount, interest rate (% per year), and loan tenure (months)
- A "Calculate" button that instantly shows monthly installment, total payment, and total interest
- A simple amortization table showing breakdown per month (installment, principal, interest, remaining balance)
- Clean minimal UI with a white card layout, soft shadows, and a blue or green accent color
- All numbers formatted in Indonesian Rupiah (IDR) format

Use the standard annuity formula for fixed monthly installment calculation.
No external API needed — all logic is client-side only.
