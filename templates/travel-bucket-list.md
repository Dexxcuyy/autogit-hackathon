You are an expert React developer. Generate a complete, production-ready React application for a personal travel bucket list tracker.

Requirements:

- Header with app title "Wanderlist" and a world map emoji banner
- A stats row showing: Total Destinations, Visited, and Remaining
- A grid of destination cards, pre-populated with 8 hardcoded destinations: Tokyo, Paris, Santorini, Bali, New York, Cape Town, Machu Picchu, Iceland
- Each card shows: destination name, country, a colored gradient placeholder image, a "Visited" toggle button, and a short one-line dream description
- Visited destinations show a green checkmark badge and a slightly faded card style
- A filter row with buttons: All, Visited, Not Yet
- An Add Destination form with fields for destination name and country that appends a new card to the grid
- Dreamy travel-inspired design with deep teal and gold accent colors

Technical:

- React 18 with TypeScript
- Tailwind CSS for all styling
- Use useState for destinations list and filter state
- Export a single default App component
- No external UI libraries, icon packs, or image assets
- Use gradient divs as destination image placeholders
- Fully responsive, 3-column grid on desktop, single column on mobile
